Elementos textuais são usados para organizar blocos ou seções de conteúdos postos no body, importante para acessibilidade e SEO, esses elementos identificam o proposito ou estrutura do conteúdo

HTML
Linguagem de marcação usada para criar páginas web.
CSS
Estiliza e define o layout das páginas HTML.
JavaScript
Linguagem de programação que adiciona interatividade às páginas web.
